This page is created by loading a file created by Consul Template.

Consul template, when run in daemon mode, will watch a key or set of keys and will update values in the template on the fly. This usually only takes milliseconds to reload and can be a powerful way of updating configuration values in application files without making them aware they are getting data from Consul.

If you're wondering why not just read the data directly from Consul, it's because this is demonstrating The actual consul-template application in action.
